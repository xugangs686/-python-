**#循环升旗**：

```
i = 0
while(1):
    time.sleep(0.4)
    t2.clear()
    t2.hideturtle()
   
    t.tracer(0)
    china_flag(-400, -400 + i * 10)
    t.tracer(10.1)
    i += 1
    if(i == 60):
        break
```

这个循环每次清除之前的绘图，绘制新的国旗位置，使其看起来像是在升起。`i` 用于控制国旗的高度，循环进行 60 次后停止。
